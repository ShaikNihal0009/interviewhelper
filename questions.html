1. What are Python decorators and how are they used?
A decorator in Python is a function that modifies the behavior of another function. It is often used for logging, authentication, etc.
2. Explain the difference between list and tuple in Python.
Lists are mutable while tuples are immutable. Lists use more memory and are slower than tuples.
